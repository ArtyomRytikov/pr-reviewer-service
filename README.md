# Pr-reviewer-service
Микросервис для автоматического назначения ревьюверов на Pull Request'ы внутри команд.

## Описание микросервиса
Был реализован сервис, который назначает ревьюеров на PR из команды автора, позволяет выполнять переназначение ревьюверов и получать список PR’ов, назначенных конкретному пользователю, а также управлять командами и активностью пользователей. После merge PR изменение состава ревьюверов запрещено.
## Запуск сервиса

### Требования
- Клонируйте репозиторий
- Docker Desktop (https://www.docker.com/products/docker-desktop)

### Список команд
Чтобы ознакомиться со списком команд запустите в терминале: make help
### Запуск
Запускается с помощью команд:
- "docker-compose up --build" - сборка + запуск
- "make docker-up"
- сначала производим сборку "docker-compose up build" и потом запускаем "docker-compose up"
