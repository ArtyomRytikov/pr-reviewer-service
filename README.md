# Pr-reviewer-service
Микросервис для автоматического назначения ревьюверов на Pull Request'ы внутри команд.

## Старт

### Требования
- Docker Desktop (https://www.docker.com/products/docker-desktop)

### Список команд
Чтобы ознакомиться со списком команл запустите в терминале: make help

## Дополнительный задания 
В качетсве дополнительных заданий были сделаны:
1)Добавить простой эндпоинт статистики (например, количество назначений по пользователям и/или по PR).
2)Провести нагрузочное тестирование полученного решения и приложить краткие результаты тестирования к решению.
