# Pr-reviewer-service
Микросервис для автоматического назначения ревьюверов на Pull Request'ы внутри команд.

## Старт

### Требования
- Docker Desktop (https://www.docker.com/products/docker-desktop)

### Список команд
Чтобы ознакомиться со списком команд запустите в терминале: make help
### Запуск
Запускается с помощью команд "docker-compose up --build" - сборка плюс запуск или "make docker-up" либо же сначала производим сборку "docker-compose up build" и потом запускаем "docker-compose up"

## Дополнительный задания 
В качетсве дополнительных заданий были сделаны:
- Добавить простой эндпоинт статистики (например, количество назначений по пользователям и/или по PR).
- Провести нагрузочное тестирование полученного решения и приложить краткие результаты тестирования к решению.
